A simple utility library for deeply cloning objects and arrays.

## Usage

#### deepClone(item)

Returns clone of given object or array

    import deepClone from "object-cloner"

    let obj = {group: 1, user: {name: "eedris", id: 3}}
    let objClone = deepClone(obj)
